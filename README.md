# lambda-edge-server

[AWS Lambda@Edge](https://aws.amazon.com/lambda/edge) handler emulator (HTTP server).

## Dependencies

- [Node.js](https://nodejs.org)

## Developers

### CLI options

Run [ESLint](https://eslint.org/) on project sources:

    $ npm run lint

Run [Mocha](https://mochajs.org) integration tests:

    $ npm run test
